I am content loaded from "examples/2.html" through an AJAX request.
A "closing link" should be provided in this response, since the response text fills (overwrites) the entire window, and thus any closing links it may have contained. To avoid this, ajax content can be "targeted" within a preconstructed window -- demonstrated in example 3b.

By default, "close triggers" (which hide the window) are assigned to element(s) with class jqmClose. If you want to use a different class name, pass the closeClass parameter. See the documentation for instructions.
